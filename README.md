# Shop Hopper

**Shop Hopper** is a modern Android e-commerce application built with **Kotlin** and **Jetpack Compose**, showcasing a scalable, production-ready architecture and modern Android development best practices. The app delivers a seamless shopping experience with secure authentication, product discovery, cart management, order tracking, and an intuitive Material Design 3 interface.

## 🛠 Tech Stack

* **100% Kotlin**
* **Jetpack Compose** for modern declarative UI
* **Material Design 3 (Material You)** with Dynamic Color support
* **MVVM + Clean Architecture**
* **Hilt** for dependency injection
* **Kotlin Coroutines & Flow** for asynchronous programming
* **Retrofit** for REST API integration
* **DataStore** for local data persistence
* **Navigation Compose**
* **Coil** for image loading

## ✨ Features

* 🔐 Secure authentication (Login, Register, Forgot Password)
* 🛍 Browse products by categories
* 🔎 Powerful product search with sorting and filtering
* ❤️ Wishlist/Favorites management
* 🤖 Personalized product recommendations
* 🛒 Shopping cart with complete checkout flow
* 🎉 Offers and discounts section
* 📦 Order tracking and order history
* 📍 Multiple shipping addresses
* 💳 Secure payment process
* 👤 User profile management
* 📱 Clean, responsive, and modern UI built with Jetpack Compose
